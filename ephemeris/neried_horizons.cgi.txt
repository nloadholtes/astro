<table cellpadding="0" cellspacing="0" border="0" width="730">
<tr><td align="left">
This tool provides a web-based <em>limited</em> interface to
<a href="/?horizons">JPL's HORIZONS system</a>
which can be used to generate ephemerides for solar-system bodies.
Full access to <a href="/?horizons">HORIZONS</a> features is available
via the primary <a href="/?horizons#telnet">telnet interface</a>.
<a href="/?horizons_news">HORIZONS system news</a>
shows recent changes and improvements.
A <a href="/?horizons_tutorial">web-interface tutorial</a> is available
to assist new users.
</td></tr>
</table>
